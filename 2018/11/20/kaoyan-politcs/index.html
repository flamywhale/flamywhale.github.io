一、马克思主义

马哲基本矛盾整理

  • 私有制商品经济的基本矛盾是私人劳动和社会劳动的矛盾。
  • 资本主义的基本矛盾是生产资料资本主义私人占有和生产社会化之间的矛盾。

马哲本质整理

  • 资本主义所有制
    • 资本家凭借对生产资料的占有，在等价交换原则的掩盖下，雇佣工人从事劳动，占有雇佣工人的剩余价值。
  • 资本
    • 资本是可以带来剩余价值的价值；资本不是物，是资本主义社会的生产关系；资本在资本主义生产过程中采取生产资料和劳动力两种形态。
  • 资本积累
    • 资本家不断利用无偿占有的工人创造的剩余价值，来扩大自己的资本规模，进一步扩大和加强对工人的剥削和统治。
  • 资本主义工资
    • 劳动力的价值或者价格
  • 资本主义经济危机的本质特征是生产相对过剩。
  • 垄断
    • 通过联合来操纵并控制商品生产和销售市场，操纵垄断价格，以攫取高额垄断利润。
  • 垄断资本的实质
    • 获取垄断利润
  • 资本主义发生变化
    • 人类社会发展一般规律和资本主义经济规律作用的结果。

马哲源泉整理

  • 社会的根本矛盾是社会发展的根本动力。
  • 抽象劳动是价值的唯一源泉。
  • 剩余价值产生的唯一源泉是雇佣劳动者的剩余劳动。
  • 劳动力商品的使用价值是价值的源泉。
    • 它在消费过程中能够创造新的价值，而且这个价值别劳动力本身的价值更大。
  • 资本积累的源泉是剩余价值。
    • 资本积累规模的大小取决于对工人的剥削程度、劳动生产率的高低、所用资本和所费资本之间的差额以及资本家垫付资本的大小。
  • 资本主义社会失业现象的根源是资本积累。
  • 垄断利润归根到底来自无产阶级和其他劳动人民所创造的剩余价值。

马哲关系整理

  • 资本主义所有制
    • 所有制是所有权的基础。
    • 所有制决定所有权。
    • 所有权是是所有制的法律形态，反映经济关系的意志关系（由经济关系决定）
  • 剩余价值与利润
    • 剩余价值是利润的本质。
    • 利润是剩余价值的转化形式。

马哲特性整理

  • 物质的唯一特性是客观实在性。
  • 运动是物质的存在方式和根本属性。
  • 时间和空间是物质运动的存在形式。
  • 精神与物质具有同一性，可以相互转化。
  • 联系的特性：客观性、多样性、普遍性、条件性。
  • 实践具有直接现实性、自觉能动性和社会历史性。社会历史性表明实践是具体的，是历史的。
  • 意识的能动性主要表现在：
    • 意识活动具有目的性和计划性。
    • 意识活动具有创造性。
    • 意识活动具有指导实践改造客观规律的作用。
    • 意识活动具有调控人的行为和生理活动的作用。
  • 规律具有客观性，人既不能创造，也不能改造规律。但是，人可以在实践中认识或发现规律，可以利用合规律的认识指导实践，还可以改变规律发生作用的条件和形式。
  • 规律是事物联系和发展过程中所固有的本质的、必然的、稳定的联系。
  • 价值的基本特性：客观性、主体性、社会历史性、多维性。
  • 生产力具有客观现实性和社会历史性。
  • 生产关系具有客观性和历史性。

（一）马克思主义是关于无产阶级和人类解放的科学

  • 德国古典哲学、英国古典政治经济学、英法两国的空想社会主义为马克思主义提供了直接的理论来源。
  • 细胞学说、能量守恒与转化定律、生物进化论为马克思主义的产生提供了自然科学基础。
  • 《德意志意识形态》第一次比较系统地阐述了历史唯物主义基本原理。
  • 人民性是马克思主义最鲜明的品格。

（二）世界的物质性和发展规律

  • 意识可以通过实践转化为物质。
  • 实践是使物质世界分化为自然界与人类社会的历史前提，又是使自然界与人类社会统一起来的现实基础。
  • 人是从事物质生产活动和其他一切社会活动的主体。
  • 人可以创造出世界上原来没有的东西。
  • 否定之否定规律
    • 内容：事物的自我完善和发展的过程。
    • 形式：螺旋式上升和波浪式前进的过程。
  • 现象与本质是同一的，可以相互转化。
  • 事物发展过程中的度是事物保持自己质的稳定性的数量界限。
  • 把握事物的可能性，要注意区分好的可能性和坏的可能性、可能性和不可能性、现实的可能性和抽象的可能性。
  • 内容是一切要素的总和，是事物存在的基础；形式是把诸要素统一起来的结构或表现内容的方式。
  • 区分质是认识量的前提，考察量是认识质的深化。
  • 矛盾双方不可调和。
  • 矛盾解决的形式：矛盾一方克服另一方；矛盾双方同归于尽；矛盾双方形成协同运动的新形式；矛盾双方融合成一个新事物。
  • 矛盾的普遍性和特殊性，即一般和个别、共性和个性、绝对和相对的关系，它们既有区别，又有联系。（“先试点再推广”）

  • “个别一定与一般相连而存在。任何个别（不论怎样）都是一般。 ——列宁”

  • “扬弃”是连续性（扬）和非连续性（弃）的统一，是联系和发展的统一，包含着肯定的否定。

（三）实践与认识及其发展规律

  • 实践具有直接现实性、自觉能动性、社会历史性。
  • 客体 ≠ 客观实在
  • 实践是人独有的活动。
  • 割裂感性认识和理性认识的统一就会导致唯理论和经验论。
  • 在真理标准问题上坚持辩证法就是要坚持实践标准的确定性与不确定性的统一：
    • 确定性：杜绝唯心主义、怀疑主义、相对主义
    • 不确定性：教条主义和独断论错误

  • “必须把人的全部实践——作为真理的标准。 ——列宁”

  • 实践能够检验一切认识。
  • 实践检验和证明真理是一个过程。
  • 真理和谬误有确定的界限！在确定的对象和范围内，真理和谬误的对立是绝对的；但超出了这个范围，二者的对立就是相对的，它们在一定条件下可以相互转化。
  • 我们所获得的认识总是具体的、实践的认识。
  • 价值是在主体的需要和利益与客体的属性或者性能的关系中发生和形成的。
  • 客观事物的某种属性能否具有价值及具有何种价值要以人的需要为基准。
  • 主客体之间的价值关系是主体在实践基础上确立的同客体之间的创造性关系。
  • 价值尺度必须以真理为前提。

（四）人类社会及其发展规律

  • 社会存在指的是社会物质生活条件，是社会生活的物质方面，包括物质生产方式、自然地理环境和人口因素。
  • 人人创造历史 ======错误===
  • 人们自己创造历史 ===正确===
  • 人的本质是一切社会关系的总和。
  • 生产关系是生产的社会形式，是社会关系中最基本的关系，是人们在物质生产过程中形成的不以人的意志为转移的经济关系；具有客观性。
  • 生产方式是劳动者和劳动资料结合的特殊方式，是社会发展的基础，是社会历史发展的决定力量。
  • 社会意识具有相对独立性，即它在反映社会存在的同时，还有自己特有的发展形势和规律。
  • 生产力是生产的物质内容。
  • 经济基础是社会的“骨骼系统”，上层建筑是社会的“血肉系统”。

（五）资本主义的本质及规律

  • 市场价格围绕商品的生产价格上下波动（平均利润产生后），最初是围绕价值上下波动。
  • 具体劳动生产商品的使用价值，抽象劳动创造商品的价值。
  • 商品一定是劳动产品。
  • 价值是交换价值的基础，交换价值是价值的表现形式，使用价值是价值和交换价值的物质承担者。
  • 劳动力的价值是由生产、发展、维持和延续劳动力所必需的生活必需品的价值决定的。
  • 充当贮藏手段的货币必须是足值的金属货币。
  • 货币的流通手段 - 现购（一手交钱，一手交货）
  • 货币的支付手段 - 清偿债务、延期支付（钱货分离）
  • 理解政治经济学的枢纽是劳动二重性理论
  • 劳动力成为商品标志着简单商品生产发展到资本主义商品生产的新阶段
  • 资本主义简单再生产的实质是物质资料再生产与资本主义生产关系再生产的统一
  • 资本循环：购买 → 生产 → 售卖
  • 资本主义经济危机的本质特征是生产过剩，经济危机爆发的根本原因是资本主义的基本矛盾；资本主义的基本矛盾是生产资料资本主义私人占有和生产社会化之间的矛盾。
  • 相对剩余价值的获得是靠全社会劳动生产率的提高
  • 资本主义再生产的特点是扩大再生产
  • 社会再生产的核心问题是社会总产品的实现，即社会总产品的价值补偿与实物替换
  • 利润掩盖了剩余价值和可变资本的关系。
  • 当利润转化为平均利润时，商品价值就转化为生产价格。
  • 利润转化为平均利润时，进一步掩盖了资本主义的剥削关系。

（六）资本主义的发展及其趋势

  • 金融资本形成的主要途径有金融联系、资本参与和人事参与。
  • 国家垄断资本主义中，微观规制的类型主要有反托拉斯法、公共事业规制和社会经济规制。
  • 法人资本所有制在性质上是一种基于资本雇佣劳动的垄断资本集体所有制。
  • 科学技术革命和生产力的发展，是资本主义变化的根本推动力量。
  • 资本主义的新变化
变化方面 历程
生产资料所有制 个体资本所有制 -> 私人股份资本所有制 -> 国家资本所有制 -> 法人资本所有制
劳资关系和分配关系 职工参与决策、终身雇佣、职工持股、社会福利制度的普及化
社会阶层和阶级结构 资本家地位变化、高级职业经理、知识型和服务型劳动者数量增加
经济调节机制和经济危机形态 更恐怖了
政治制度 政治制度多元化，公民权利扩大；公民谋求自身利益；改良主义政党影响扩大

（七）社会主义的发展及其规律

  • 科学社会主义问世的标志是《共产党宣言》的发表
  • 建立无产阶级证券的第一次伟大尝试是1871年的巴黎公社革命

二、毛中特

  • 1938年，毛泽东在党的六届六中全会上作的题为《论新阶段》的政治报告中，最先提出了“马克思主义中国化”这个命题。
  • 毛泽东思想初步形成的标志是提出并阐述农村包围城市、武装夺取政权的思想。（第一次国内革命战争和土地革命时期）
  • 新民主主义理论的系统阐释标志着毛泽东思想达到成熟。（抗日战争时期）

新民主主义革命理论

  • 1939年，毛泽东在《中国革命和中国共产党》一文中，第一次提出了新民主主义革命的科学概念和总路线的内容。
  • 1948年，毛泽东在《在晋绥干部会议上的讲话》完整地表述了总路线的内容，指出：“无产阶级领导的，人民大众的，反对帝国主义、封建主义和官僚资本主义的革命，这就是中国新民主主义的革命，这就是中国共产党在当前历史阶段的总路线和总政策。”
  • 《在晋绥干部会议上的讲话》提出了完整的土地改革的总路线，即依靠贫雇农，团结中农，有步骤地、有分别地消灭封建剥削制度，发展农业生产。（经济）
  • 无产阶级的领导权问题是中国革命的核心问题，因此区分新旧民主主义革命的根本标志是领导权掌握在无产阶级还是资产阶级手中。
  • 无产阶级掌握领导权是中国革命取得胜利的根本保证。
  • 无产阶级领导权实现的关键是建立广泛的以工农联盟为基础的统一战线。
  • 无产阶级领导权实现的基本策略是保证政治上、组织上的独立性。
  • 无产阶级领导权实现的坚强支柱是革命武装力量。
  • 不同时期的统一战线的根本经验是处理好和资产阶级的关系。
  • 建立统一战线的必要性：
    • 中国半殖民地半封建社会的阶级状况
    • 中国革命的长期性、残酷性及其发展的不平衡性

新民主主义社会

  • 政治纲领：推翻帝国主义和封建主义的统治，建立一个无产阶级领导的、以工农联盟为基础的、各革命阶级联合专政的新民主主义的共和国。
  • 政治纲领规定的新民主主义国家
    • 国体：各革命阶级联合专政
    • 政体：民主集中制的人民代表大会制度
    • 人民当家作主是国家制度的核心内容和基本准则
  • 随着土地改革基本完成（1952年底），工人阶级和资产阶级的矛盾逐步成为国内的主要矛盾。

社会主义建设道路初步探索

  • 1956年，三大改造的基本完成标志着社会主义制度在我国初步确立。
  • 《论十大关系》标志着党探索中国社会主义建设道路的良好开端。
  • 走中国工业化道路的要求
    1. 必须明确战略目标和战略要求。
    2. 必须采取正确的经济建设方针。
    3. 必须发展科学技术和文化教育。
    4. 必须重视知识分子工作。
    5. 必须调整和完善所有制结构。
    6. 必须积极探索符合我国情况的经济体制和运行机制。

邓小平理论

主要内容在中共代表大会中整理。

  • 回答的基本问题是：什么是社会主义？怎样建设社会主义？
  • 和平与发展的时代背景是邓小平理论形成的时代背景。
  • 社会主义建设的经验教训是邓小平理论形成的历史根据。
  • 改革开放和现代化建设的实践是邓小平理论形成的现实依据。
  • 邓小平理论形成轮廓是在中共十三大中。
  • 邓小平理论走向成熟是在1992年的南方谈话中。
  • 邓小平理论被写入党章是在中共十四大。

毛主席著作整理

  • 1935年12月，毛泽东作了《论反对日本帝国主义的策略》的报告，它阐明党的抗日民族统一战线的新政策，批判党内的关门主义和对于革命的急性病，系统地解决了党的政治路线上的问题。
  • 毛泽东第一次把党的建设称为伟大工程的著作是《<共产党人>发刊词》。
  • 《<共产党人>发刊词》提出了新民主主义革命的三大法宝：统一战线、武装斗争和党的建设。
  • 1938年，毛泽东发表了《论持久战》，系统地阐明了持久抗战的总方针。
  • 抗日战争时期，毛泽东在《战争和战略问题》中把“经过长期武装斗争，先占乡村，后占城市，最终取得胜利”，作为革命道路确立下来。
  • 1945年，毛泽东在七大中作的《论联合政府》的政治报告中，首次对党的优良作风作了简明概括。
  • 1956年，毛泽东提出了《论十大关系》。
    • 开始探索社会主义建设道路的标志。
    • 马克思主义和中国具体实际第二次结合的开始。
    • 基本方针：“努力把党内党外、国内国外的一切积极的因素，直接的、间接的积极因素全部调动起来”，为社会主义建设服务。
  • 毛泽东在《关于正确处理人民内部矛盾的问题》中指出，我们国家政治生活的主题是正确处理人民内部矛盾。
  • 毛泽东提出了“两参一改三结合”。
  • 1957年7月，毛泽东撰写的《一九五七年夏季的形式》中提出，要造成一个又有集中又有民主，又有纪律又有自由，又有统一意志，又有个人心情舒畅、生动活泼，那样一种政治局面的思想。（“六又”政治局面）

  • 中共七大（1945年）

    • 将毛泽东思想定为党的一切工作的方针。

    • 经济建设方针：既反保守又反冒进，在综合平衡中稳步前进

（五）明大德守公德严私德

  • 道德起源的第一个历史前提是劳动。
  • 道德在本质上是知行合一的。
  • 贯穿中国革命道德始终的一根红线是全心全意为人民服务。
  • “吾日三省吾身”属于中国传统道德修养方法中的省察克治。

（六）尊法学法守法用法

  • 决定法律本质、内容和发展方向的根本因素是生产方式。
  • 法律运行的关键性（起始性）环节是法律制定。
  • 法律适用是指国家司法机关及公职人员依照法定职权和程序适用法律处理案件的专门活动。
  • 守法是法律实施和实现的基本途径。
  • 狭义上的法律执行的主体是国家行政机关。
  • 人民代表大会制度是中国的政体，是中国社会主义民主政治最鲜明的特点。
  • 全面依法治国格局
    • 科学立法
      • 以完善以宪法为核心的中国特色社会主义法律体系，加强宪法实施为目标。
    • 严格执法
      • 以深入推进依法行政，加快建设法治政府为目标。
    • 全民守法
      • 以增强全民法治观念，推进法治社会建设为目标。
    • 公正司法
  • 全面依法治国的根本目的是实现人民幸福，尊重和保障人权。
  • 党的领导是中国特色社会主义最本质的特征，是社会主义法治最根本的保证。
  • 平等是社会主义法律的基本属性，是社会主义法治的基本要求。
  • 权利保障的关键环节是行政保护。
  • 强调社会的物质生活条件对权利的制约和决定作用，是马克思主义权利观和其他权利观的根本区别。
  • 宪法有规范、引领、推动、保障作用。
  • 国体：人民民主专政；政体：人民代表大会制度。
  • 法律至上性表现为法律的普遍适用性、优先适用性和不可违抗性。
  • 权利的法律保障包括宪法保障、立法保障、行政保护和司法保障。
  • 法律义务的特性。
