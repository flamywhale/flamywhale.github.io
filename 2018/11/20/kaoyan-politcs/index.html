一、马克思主义

马哲基本矛盾整理

  • 私有制商品经济的基本矛盾是私人劳动和社会劳动的矛盾。
  • 资本主义的基本矛盾是生产资料资本主义私人占有和生产社会化之间的矛盾。

马哲本质整理

  • 资本主义所有制
    • 资本家凭借对生产资料的占有，在等价交换原则的掩盖下，雇佣工人从事劳动，占有雇佣工人的剩余价值。
  • 资本
    • 资本是可以带来剩余价值的价值；资本不是物，是资本主义社会的生产关系；资本在资本主义生产过程中采取生产资料和劳动力两种形态。
  • 资本积累
    • 资本家不断利用无偿占有的工人创造的剩余价值，来扩大自己的资本规模，进一步扩大和加强对工人的剥削和统治。
  • 资本主义工资
    • 劳动力的价值或者价格
  • 资本主义经济危机的本质特征是生产相对过剩。
  • 垄断
    • 通过联合来操纵并控制商品生产和销售市场，操纵垄断价格，以攫取高额垄断利润。
  • 垄断资本的实质
    • 获取垄断利润
  • 资本主义发生变化
    • 人类社会发展一般规律和资本主义经济规律作用的结果。

马哲源泉整理

  • 社会的根本矛盾是社会发展的根本动力。
  • 抽象劳动是价值的唯一源泉。
  • 剩余价值产生的唯一源泉是雇佣劳动者的剩余劳动。
  • 劳动力商品的使用价值是价值的源泉。
    • 它在消费过程中能够创造新的价值，而且这个价值别劳动力本身的价值更大。
  • 资本积累的源泉是剩余价值。
    • 资本积累规模的大小取决于对工人的剥削程度、劳动生产率的高低、所用资本和所费资本之间的差额以及资本家垫付资本的大小。
  • 资本主义社会失业现象的根源是资本积累。
  • 垄断利润归根到底来自无产阶级和其他劳动人民所创造的剩余价值。

马哲关系整理

  • 资本主义所有制
    • 所有制是所有权的基础。
    • 所有制决定所有权。
    • 所有权是是所有制的法律形态，反映经济关系的意志关系（由经济关系决定）
  • 剩余价值与利润
    • 剩余价值是利润的本质。
    • 利润是剩余价值的转化形式。

马哲特性整理

  • 物质的唯一特性是客观实在性。
  • 运动是物质的存在方式和根本属性。
  • 时间和空间是物质运动的存在形式。
  • 精神与物质具有同一性，可以相互转化。
  • 联系的特性：客观性、多样性、普遍性、条件性。
  • 实践具有直接现实性、自觉能动性和社会历史性。社会历史性表明实践是具体的，是历史的。
  • 意识的能动性主要表现在：
    • 意识活动具有目的性和计划性。
    • 意识活动具有创造性。
    • 意识活动具有指导实践改造客观规律的作用。
    • 意识活动具有调控人的行为和生理活动的作用。
  • 规律具有客观性，人既不能创造，也不能改造规律。但是，人可以在实践中认识或发现规律，可以利用合规律的认识指导实践，还可以改变规律发生作用的条件和形式。
  • 规律是事物联系和发展过程中所固有的本质的、必然的、稳定的联系。
  • 价值的基本特性：客观性、主体性、社会历史性、多维性。
  • 生产力具有客观现实性和社会历史性。
  • 生产关系具有客观性和历史性。

（一）马克思主义是关于无产阶级和人类解放的科学

  • 德国古典哲学、英国古典政治经济学、英法两国的空想社会主义为马克思主义提供了直接的理论来源。
  • 细胞学说、能量守恒与转化定律、生物进化论为马克思主义的产生提供了自然科学基础。
  • 《德意志意识形态》第一次比较系统地阐述了历史唯物主义基本原理。
  • 人民性是马克思主义最鲜明的品格。

（二）世界的物质性和发展规律

  • 意识可以通过实践转化为物质。
  • 实践是使物质世界分化为自然界与人类社会的历史前提，又是使自然界与人类社会统一起来的现实基础。
  • 人是从事物质生产活动和其他一切社会活动的主体。
  • 人可以创造出世界上原来没有的东西。
  • 否定之否定规律
    • 内容：事物的自我完善和发展的过程。
    • 形式：螺旋式上升和波浪式前进的过程。
  • 现象与本质是同一的，可以相互转化。
  • 事物发展过程中的度是事物保持自己质的稳定性的数量界限。
  • 把握事物的可能性，要注意区分好的可能性和坏的可能性、可能性和不可能性、现实的可能性和抽象的可能性。
  • 内容是一切要素的总和，是事物存在的基础；形式是把诸要素统一起来的结构或表现内容的方式。
  • 区分质是认识量的前提，考察量是认识质的深化。
  • 矛盾双方不可调和。
  • 矛盾解决的形式：矛盾一方克服另一方；矛盾双方同归于尽；矛盾双方形成协同运动的新形式；矛盾双方融合成一个新事物。
  • 矛盾的普遍性和特殊性，即一般和个别、共性和个性、绝对和相对的关系，它们既有区别，又有联系。（“先试点再推广”）

  • “个别一定与一般相连而存在。任何个别（不论怎样）都是一般。 ——列宁”

  • “扬弃”是连续性（扬）和非连续性（弃）的统一，是联系和发展的统一，包含着肯定的否定。

（三）实践与认识及其发展规律

  • 实践具有直接现实性、自觉能动性、社会历史性。
  • 客体 ≠ 客观实在
  • 实践是人独有的活动。
  • 割裂感性认识和理性认识的统一就会导致唯理论和经验论。
  • 在真理标准问题上坚持辩证法就是要坚持实践标准的确定性与不确定性的统一：
    • 确定性：杜绝唯心主义、怀疑主义、相对主义
    • 不确定性：教条主义和独断论错误

  • “必须把人的全部实践——作为真理的标准。 ——列宁”

  • 实践能够检验一切认识。
  • 实践检验和证明真理是一个过程。
  • 真理和谬误有确定的界限！在确定的对象和范围内，真理和谬误的对立是绝对的；但超出了这个范围，二者的对立就是相对的，它们在一定条件下可以相互转化。
  • 我们所获得的认识总是具体的、实践的认识。
  • 价值是在主体的需要和利益与客体的属性或者性能的关系中发生和形成的。
  • 客观事物的某种属性能否具有价值及具有何种价值要以人的需要为基准。
  • 主客体之间的价值关系是主体在实践基础上确立的同客体之间的创造性关系。
  • 价值尺度必须以真理为前提。

（四）人类社会及其发展规律

  • 社会存在指的是社会物质生活条件，是社会生活的物质方面，包括物质生产方式、自然地理环境和人口因素。
  • 人人创造历史 ======错误===
  • 人们自己创造历史 ===正确===
  • 人的本质是一切社会关系的总和。
  • 生产关系是生产的社会形式，是社会关系中最基本的关系，是人们在物质生产过程中形成的不以人的意志为转移的经济关系；具有客观性。
  • 生产方式是劳动者和劳动资料结合的特殊方式，是社会发展的基础，是社会历史发展的决定力量。
  • 社会意识具有相对独立性，即它在反映社会存在的同时，还有自己特有的发展形势和规律。
  • 生产力是生产的物质内容。
  • 经济基础是社会的“骨骼系统”，上层建筑是社会的“血肉系统”。

（五）资本主义的本质及规律

  • 市场价格围绕商品的生产价格上下波动（平均利润产生后），最初是围绕价值上下波动。
  • 具体劳动生产商品的使用价值，抽象劳动创造商品的价值。
  • 商品一定是劳动产品。
  • 价值是交换价值的基础，交换价值是价值的表现形式，使用价值是价值和交换价值的物质承担者。
  • 劳动力的价值是由生产、发展、维持和延续劳动力所必需的生活必需品的价值决定的。
  • 充当贮藏手段的货币必须是足值的金属货币。
  • 货币的流通手段 - 现购（一手交钱，一手交货）
  • 货币的支付手段 - 清偿债务、延期支付（钱货分离）
  • 理解政治经济学的枢纽是劳动二重性理论
  • 劳动力成为商品标志着简单商品生产发展到资本主义商品生产的新阶段
  • 资本主义简单再生产的实质是物质资料再生产与资本主义生产关系再生产的统一
  • 资本循环：购买 → 生产 → 售卖
  • 资本主义经济危机的本质特征是生产过剩，经济危机爆发的根本原因是资本主义的基本矛盾；资本主义的基本矛盾是生产资料资本主义私人占有和生产社会化之间的矛盾。
  • 相对剩余价值的获得是靠全社会劳动生产率的提高
  • 资本主义再生产的特点是扩大再生产
  • 社会再生产的核心问题是社会总产品的实现，即社会总产品的价值补偿与实物替换
  • 利润掩盖了剩余价值和可变资本的关系。
  • 当利润转化为平均利润时，商品价值就转化为生产价格。
  • 利润转化为平均利润时，进一步掩盖了资本主义的剥削关系。

（六）资本主义的发展及其趋势

  • 金融资本形成的主要途径有金融联系、资本参与和人事参与。
  • 国家垄断资本主义中，微观规制的类型主要有反托拉斯法、公共事业规制和社会经济规制。
  • 法人资本所有制在性质上是一种基于资本雇佣劳动的垄断资本集体所有制。
  • 科学技术革命和生产力的发展，是资本主义变化的根本推动力量。
  • 资本主义的新变化
变化方面 历程
生产资料所有制 个体资本所有制 -> 私人股份资本所有制 -> 国家资本所有制 -> 法人资本所有制
劳资关系和分配关系 职工参与决策、终身雇佣、职工持股、社会福利制度的普及化
社会阶层和阶级结构 资本家地位变化、高级职业经理、知识型和服务型劳动者数量增加
经济调节机制和经济危机形态 更恐怖了
政治制度 政治制度多元化，公民权利扩大；公民谋求自身利益；改良主义政党影响扩大

（七）社会主义的发展及其规律

  • 科学社会主义问世的标志是《共产党宣言》的发表
  • 建立无产阶级证券的第一次伟大尝试是1871年的巴黎公社革命

三、中国近代史纲要

毛主席著作整理

史纲中的“第一次XXX”整理

  • 近代中国“正眼看世界”第一人是林则徐。
  • 中国近代史上人民群众第一次大规模的反侵略武装斗争是三元里人民的抗英斗争。
  • 中国民族资产阶级登上政治舞台的第一次表演是戊戌维新运动。
  • 中共二大第一次提出了反帝反封建的民主革命纲领。
  • 中国工人运动第一次高潮起点的标志是香港海员罢工。
  • 中国第一个农民协会诞生于浙江萧山县。

史纲中的中共会议整理

中共X大

  • 中共七大
    • 在延安举行
    • 毛泽东思想诞生：以毛泽东为主要代表的中国共产党人把马克思列宁主义同中国具体实际想结合所创造的理论成果命名为毛泽东思想。
    • 将毛泽东思想定为党的一切工作的方针。
  • 中共八大
    • 经济建设方针：在综合平衡中稳步前进
    • 国内主要矛盾：人民对于经济文化迅速发展的需要同当前经济文化不能满足人民需要的状况之间的矛盾。
    • 根本任务：在新的生产关系下保护和发展生产力。
  • 中国十二大（1982年9月）
    • 建设有中国特色的社会主义
    • 改革重点从农村转向城市
    • 多层次对外开放格局形成
  • 中国十三大（1987年）
    • 社会主义初级阶段基本路线：“一个中心，两个基本点”
    • 提出社会主义初级阶段理论
    • 提出了“三步走”的发展战略
  • 中共十四大（1992年）
    • 确立了我国经济体制改革的目标是建立社会主义市场经济体制。
    • 确立了邓小平建设有中国特色社会主义理论在全党的指导地位。
  • 中共十五大（1997年）
    • 邓小平理论写入党章
    • 提出了党在社会主义初级阶段的基本纲领。
  • 中共十六大（2002年）
    • 提出了全面建设小康社会的奋斗目标。
    • “三个代表”思想写入党章。
  • 中共十七大（2007年）
    • 科学发展观写入党章。
    • 提出全面建设小康社会的新要求。
  • 中共十八大（2012年）
    • 坚持和发展中国特色社会主义。
    • 标志着中国已经进入全面建成小康社会的决定性阶段，开启了中国特色社会主义新时代。
  • 中共十九大 （2017年）
    • 政治建设摆在首位
    • 确立习近平新时代中国特色社会主义思想的历史地位。

X届X中全会

  • 中共七届二中全会（1949年3月）
    • 工作重心：乡村转移到城市
    • 两个转变：
      • 农业国转变为工业国；
      • 新民主主义社会转变为社会主义社会；
    • 两个务必：
      • 务必使同志们继续地保持谦虚、谨慎、不骄不躁的作风
      • 务必使同志们继续地保持艰苦奋斗的作风
  • 中共十一届三中全会
    • 进入改革开放和社会主义现代化建设的历史新时期的标志。
